# How Your BF/GF Is Cheating On You

A fun and straightforward website where you go through 50 red flags and check off whichever ones apply to your relationship. If you check more than 3, the chances that your partner is cheating on you are guaranteed.

No sign up. No install. Just the truth.

## Live Site

https://how-your-bf-gf-is-cheating-on-you.vercel.app/

## How It Works

1. Open the website
2. Go through the list of 50 red flags
3. Check off every one that applies to your relationship
4. If you select more than 3 the verdict is in

## Built With

- HTML
- CSS
- JavaScript

No frameworks. No dependencies. Just vanilla web.

## Running It Locally

Since this is a plain HTML, CSS and JavaScript project there is nothing to install or set up.

Just clone the repo and open the index.html file in your browser.

```bash
git clone https://github.com/xodo2fast4u/how-your-bf-gf-is-cheating-on-you.git
cd how-your-bf-gf-is-cheating-on-you
```

Then open index.html in your browser and you are good to go.
